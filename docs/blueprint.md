# **App Name**: Blank Slate Login

## Core Features:

- User Authentication: Implements user login functionality with current security practices.
- Blank Input Fields: Provides completely empty fields for username and password.
- Submit Button: Allows users to attempt login with the provided credentials.
- Error Handling: Displays generic error messages for failed login attempts.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to create a sense of security.
- Background color: Very light indigo (#E8EAF6) to create a clean background.
- Accent color: Violet (#7986CB) for subtle accents and interactive elements.
- Body and headline font: 'Inter', a sans-serif for clarity and modern look.
- Simple, minimalist icons for input fields, such as a user icon for username and a lock icon for password.
- Clean, centered layout with input fields stacked vertically for ease of use.
- Subtle transition animations when input fields are focused.
